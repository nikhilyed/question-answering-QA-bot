# QA Bot - PDF Question Answering with IBM Watsonx AI

This project is a **Question Answering (QA) Bot** that allows users to upload a PDF document and ask questions about its content. The bot uses **IBM Watsonx Foundation Models** for natural language understanding and **LangChain** to process, split, and retrieve relevant text from documents. The interface is built using **Gradio** for a simple web-based UI.
